Northshore Logistics Database Application

How to Use

1. Run the application by stating main.py
2. On first launch, create an admin account.
3. Log in using your username and password.
4. Admin users can create additional users with different roles:
      admin
      manager
      warehouse
      driver
   This controls what each user can access in the system.



Navigation

Use the left menu to access features:
  Shipments → Add new shipment
  Shipment Updates → Update shipment status (creates history)
  Deliveries / Incidents → Add records using Order Number
  Inventory / Vehicles / Drivers → Manage operational data
  Warehouse Activity / Vehicle Utilisation → Record activity
  Tables → View all stored data
  Reports → Generate shipment report using Order Number



Data Management

You can edit records to correct mistakes (overwrites data)
You can update shipments to track progress (creates history)
You can delete individual records or full history



Important Notes

Always use Order Number, not shipment ID
Shipment ID is handled internally by the system
Updates create history, edits overwrite existing data
